# نبض 🥇 — منصة تداول الذهب الاحترافية (XAU/USD)

تطبيق ويب من صفحة واحدة (لا يحتاج خادم/باك-إند) لتحليل الذهب لحظيًا، يجمع بين:

- **تحليل فني كلاسيكي**: RSI, MACD, ATR, Bollinger Bands, EMA 20/50/200, VWAP, فيبوناتشي (أقوى ٣ مستويات)، دعوم/مقاومات، أنماط شموع يابانية مع أيقونات، أنماط فنية على الشارت (قمة/قاع مزدوج، رأس وكتفين، مثلثات).
- **Smart Money Concepts (SMC)**: هيكل السوق (BOS/CHOCH)، Order Blocks، Fair Value Gaps، مناطق السيولة.
- **محرك توصيات مبني على بوابة جودة**: لا تظهر توصية دخول فعلية إلا إذا توافقت عدة عوامل بثقة كافية — وإلا تظهر حالة "انتظار". يعرض نقطة الدخول، وقف الخسارة، وثلاثة أهداف (TP1/TP2/TP3)، ونسبة المخاطرة للعائد، ونسبة ثقة/احتمالية نجاح.
- **زر "هل أدخل الصفقة الآن؟"**: يعيد التحليل بالكامل فورًا ويعرض قرارًا واضحًا (دخول / انتظار) مع أهم الأسباب.
- **تقويم اقتصادي** لأخبار الدولار عالية التأثير، مقسّم حسب اليوم (اليوم/غدًا)، مع تحديث يدوي وتخزين مؤقت محلي لضمان الموثوقية.
- **جلسات التداول العالمية** (سيدني/طوكيو/لندن/نيويورك) مع حالة الفتح/الإغلاق والعد التنازلي.
- **إدارة مخاطر**: حاسبة حجم اللوت المقترح بناءً على رصيد الحساب ونسبة المخاطرة.
- **تحليل عميق اختياري بالذكاء الاصطناعي** (Claude): إن أضفت مفتاح Anthropic API الخاص بك من الإعدادات، يقوم التطبيق بطلب تحليل نصي معمّق يدمج الفني بالأساسي.

كل الحسابات تتم في المتصفح مباشرة (لا يوجد سيرفر)، وتعتمد على مصادر بيانات مجانية:
Binance (PAXG كبديل لسعر الذهب) أو TwelveData (سعر XAU/USD الفعلي، يتطلب مفتاح مجاني)، وForexFactory/TradingEconomics للتقويم الاقتصادي.

## ⚠️ تنويه مهم

هذا التطبيق أداة تحليلية تعليمية وليس نصيحة استثمارية. التداول في الذهب والعقود مقابل الفروقات ينطوي على مخاطر خسارة كبيرة. التوصيات والاحتمالات المعروضة مبنية على قواعد فنية مبرمجة وليست ضمانًا لأي نتيجة. راجع استراتيجيتك مع مصادر موثوقة وطبّق إدارة مخاطر صارمة دائمًا.

## التشغيل محليًا

لا يحتاج أي تثبيت — فقط افتح `index.html` في المتصفح، أو استخدم أي خادم استاتيكي بسيط:

```bash
npx serve .
# أو
python3 -m http.server 8080
```

## النشر على GitHub Pages / Netlify / Vercel / Cloudflare Pages

المشروع بالكامل عبارة عن ملفات استاتيكية (`index.html`, `manifest.json`, `sw.js`, `_headers`) — لا حاجة لأي عملية بناء (build).

### رفعه على GitHub

من داخل مجلد المشروع، نفّذ الأوامر التالية (بعد إنشاء مستودع فارغ جديد على github.com):

```bash
git init
git add .
git commit -m "نبض — منصة تداول الذهب الاحترافية v1"
git branch -M main
git remote add origin https://github.com/USERNAME/REPO_NAME.git
git push -u origin main
```

استبدل `USERNAME/REPO_NAME` باسم المستخدم واسم المستودع الخاصين بك.

### ثم النشر تلقائيًا من GitHub إلى استضافة مجانية

- **GitHub Pages**: من إعدادات المستودع → Pages → اختر الفرع `main` والمجلد الجذر (`/`) → حفظ. سيصبح الموقع متاحًا على `https://USERNAME.github.io/REPO_NAME/`.
- **Netlify**: اربط حساب Netlify بمستودع GitHub (New site from Git) — لا حاجة لأي أمر بناء (build command فارغ) ومجلد النشر هو `.` (الجذر).
- **Vercel**: استورد المستودع من Vercel Dashboard، اختر "Other" كإطار عمل (Framework Preset)، ولا حاجة لأمر بناء.
- **Cloudflare Pages**: اربط المستودع، اترك أمر البناء فارغًا ومجلد الإخراج هو `/`.

بعد الربط، أي `git push` جديد سيُحدّث الموقع المباشر تلقائيًا على أي من هذه المنصات.

## الإعدادات داخل التطبيق

من أيقونة ⚙️ الإعدادات:

1. **مصدر سعر الذهب**: PAXG عبر Binance (مجاني بدون مفتاح) أو TwelveData (سعر XAU/USD الفعلي، يتطلب مفتاح مجاني من twelvedata.com).
2. **مفتاح Anthropic API (اختياري)**: لتفعيل التحليل العميق بالذكاء الاصطناعي. يُحفظ محليًا في متصفحك فقط ويُستخدم مباشرة من المتصفح — يُنصح باستخدام مفتاح بحد إنفاق محدود من console.anthropic.com.

## بنية الملفات

```
index.html      ← التطبيق بالكامل (HTML + CSS + JS في ملف واحد)
manifest.json    ← إعدادات PWA (تثبيت التطبيق على الهاتف)
sw.js            ← service worker بسيط للعمل شبه دون اتصال
_headers         ← رؤوس HTTP (لـ Netlify)
test.html        ← صفحة اختبار مساعدة
```
